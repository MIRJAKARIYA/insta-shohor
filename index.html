How does Javascript work?

Javascript is a client side scripting language which runs in the browser.In every browser there is a javascript engine that converts our raw javascript code into machine executable code or byte code.for example chrome-V8 engine,Firefox-Spider monkey,Internet explorer-Chakra.When we load javascript code in the browser,the javascript engine takes the code and passes it into the parser.Parser checks the code line by line searching for errors and if any error is found in any line then the javascript engine stops parsing immediately and throws error in the console.If the whole code is okay then the parser produces a data structure called AST(Abstract Syntax Tree) which contains a tree representation of the javascript code.Then the engine converts the javascript code from the AST into machine code or byte code by interpreting(executing the code line by line).Now the processor can understand the byte code and it runs the code.Chrome V8 engine uses a special type of compiler called JIT(Just In Time) compiler which makes the interpreting process faster.
